Frequently Asked Questions (MToolBox v1.0)

  • Q1> Which reference genome are you using?

    RSRS.

  • Q2> Which paper should I cite?

    Calabrese C, Simone D, Diroma MA, Santorsola M, Gutt√† C, Gasparre G, Picardi E, Pesole G, Attimonelli M. MToolBox: a highly automated pipeline for heteroplasmy annotation and prioritization analysis of human mitochondrial variants in high-throughput sequencing. Bioinformatics. 2014 Nov 1;30(21):3115-7.

  • Q3> Which parameters are being used?

    We use MToolBox default parameters (UseMarkDuplicates=true, UseIndelRealigner=false).



Frequently Asked Questions (SG-ADVISER mtDNA)

  • Q1> Which paper should I cite?

    Rueda M and Torkamani A, SG-ADVISER mtDNA: A web server for mitochondrial DNA annotation with data from 200 samples of a healty aging cohort. BMC Bioinformatics 2017 18:373.

  • Q2> Why the number of variants reported in the HTML table and the REPORT file differ?

    Because in the HTML we are not displaying synonymous variants.

  • Q3> Who do I contact if my job failed or to report a bug?

    You can contact the team of developers, which consists of one individual (email under Menu>Links).

  • Q4> Can I get a copy of SG-ADVISER mtDNA software to make a local installation?

    Yes, you can. Just send an email to the developer (Menu>Links). Having said that, the whole purpose of making this web server was to avoid the local installations :-)
    Note that we offer basic web services (see help page).
